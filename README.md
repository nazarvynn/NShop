NShop
=====

#Install:

##Express
`$ npm install express --save`

##Update modules
`$ npm update`

##MongoDB
Go to MongoDB console <br />
`$ mongo` <br />
`> db.createCollection("categories")` <br />
`> db.createCollection("users")`

#Run
`$ cd NShop`<br />
`$ node app`

#Info
List of npm modules <br />
`$ npm ls`
